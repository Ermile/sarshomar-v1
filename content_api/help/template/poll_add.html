ثبت یک نظر سنجی

برای ثبت یک نظر سنجی ابتدا باید وارد سیستم شود سپس طبق الگوی زیر عمل می کنید. باید توجه داشت که بسیاری از پارامتر های ورودی طبق نوع نظر سنجی شما تنظیم می شوند به عبارت دیگر برخی از پارامتر ها در نوع خاصی از نظر سنجی استفاده می شوند و برخی در نوع دیگر

URL

https://sarshomar.com/api/poll

METHOD : POST

Input

  • نوع نظر سنجی
    که میتواند یکی از مقادیر زیر باشد
    multiple
    descriptive
    notification
    upload
    range
    String اجباری type
  • عنوان نظر سنجی (متن سوال) String اجباری title
  • گزینه های سوال. در حالت سوال multiple اجباری String اجباری answer1,answer2,...
  • نام آپلود Files اختیاری upload_name
  • توضیحاتی کامل در خصوص این نظر سنجی String اختیاری description
  • خلاصه ای از سوال جهت نمایش در شبکه های اجتماعی String اختیاری summary
  • شناسه سوال سرشاخه درختواره String اختیاری parent_tree_id
  • گزینه های سوال درختواره String اختیاری parent_tree_opt
  • حد اقل زمان پاسخگویی به نظر سنجی integer اختیاری rangetiming-min
  • حد اکثر زمان پاسخگویی به نظر سنجی اختیاری rangetiming-max
  • حد اقل حجم فایل ارسالی توسط کاربر. این مقدار در حالت آپلود فایل استفاده خواهد شد اختیاری filesize-min
  • حد اکثر حجم فایل ارسالی توسط کاربر. این مقدار در حالت آپلود فایل استفاده خواهد شد اختیاری filesize-max
  • حد اقل طول متن اختیاری textlength-min
  • حد اکثر طول متن اختیاری textlength-max
  • حد اقل عدد ورودی اختیاری numbersize-min
  • حد اکثر عدد ورودی اختیاری numbersize-max
  • حد اقل ستاره ها برای سوال ستاره ای اختیاری starsize-min
  • حد اکثر ستاره ها برای سوال ستاره ای اختیاری starsize-max
  • حد اقل گزینه های انتخابی توسط کاربر اختیاری choice-count-min
  • حد اکثر گزینه های انتخابی توسط کاربر اختیاری choice-count-max
  • مرتب سازی تصادفی سوال on|off اختیاری random_sort
  • سوال دارای گزینه های امتیاز دار on|off اختیاری score
  • سوال دارای گزینه صحیح on|off اختیاری true_answer
  • سوال دارای گزینه سایر on|off اختیاری descriptive
  • مخفی سازی نتایج سوال on|off اختیاری hidden_result
  • اجازه ثبت کامنت برای سوال on|off اختیاری comment
  • سوال مرتب سازی on|off اختیاری ordering
  • نوع پاسخگویی کاربر به گزینه ها one|multi|ordering اختیاری choicemode
  • نوع متن درخواستی از سمت کاربر اختیاری text_format
  • نوع فایل درخواستی از سمت کاربر اختیاری file_format
  • نوع انتخاب سوال محدوده ای number|star|like اختیاری rangemode
  • درخواست متن دلخواه از کاربر اختیاری text_format_custom
  • درخواست نوع خاصی فایل از کاربر اختیاری file_format_custom

Output

{
  "status": 1,
  "title": null,
  "msg": {
    "callback": "4sJhf"
  },
  "messages": {
    "true": [
      {
        "title": "Poll Successfully added",
        "element": false,
        "group": "public"
      }
    ]
  }
}

خطاهایی که در حین عملیات رخ میدهد در قسمت messages به شما نشان داده خواهد شد و در صورت موفقیت آمیز بودن عملیات شناسه نظر سنجی شما در قسمت msg->callback برای استفاده های بعدی به شما اعلام می شود.
